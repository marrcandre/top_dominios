# Baixando e configurando

- Abra um terminal:

    `Control"` + `Alt` + `t`

- Clone o repositório:

```bash
git clone https://github.com/marrcandre/top_dominios.git
```

- Entre na pasta:
  
```bash
cd top_dominios
```

- Instale as dependências:

```bash
poetry install
```

- Ative o ambiente virtual:

```bash
poetry shell
```

- Entre na pasta do projeto:

```bash
cd src
``` 

- Execute o script:

```bash
python top_dominios.py
```

- Se preferir, abra o vscode:

```bash
code .
```
